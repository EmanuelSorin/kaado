Information & Help

Welcome to Your Japanese Practice Platform

This website is designed to make learning Japanese fun and interactive! Choose from different game modes, practice kanas (hiragana and katakana), kanji, grammar, and vocabulary from JLPT levels N5 to N1. Here’s how to get started:

Main Menu

From the main menu, you can choose to:
  • Practice Japanese - Access the game modes to test your knowledge.
  • Alphabet Section - Review all the kana types and kanji available and use the search tool to quickly locate specific kanji. Kanji are organized by JLPT level (N5 through N1).

Game Modes

There are two game modes for practicing Japanese:
  • Flashcard Mode - Choose the correct answer from multiple options.
  • Typing Mode - Type the answer to match the prompt displayed.

Game Interface Overview

The game window has three main sections:

1. Options Panel

The options panel lets you customize your learning experience. Here are the features you can adjust:
  • Mode - Switch between flashcard and typing modes.
  • Practice Type - Choose what you want to study:
    • For kana, select from main kana, dakuten, etc.
    • For kanji, select by JLPT level (N5, N4, N3, etc.).
    • You can specify the range of kanji you want to practice by clicking on the title icon in this section. For example, select only kanji 1 to 20.
  • Difficulty - Adjust the difficulty level (available in flashcard mode only):
    • Easy - Shows 2 choices.
    • Medium - Shows 4 choices.
    • Hard - Shows 9 choices.
  • Guess Mode - Choose if you want to guess using:
    • Romaji - The prompt appears in romaji.
    • Kana - The prompt appears in kana. (This option is available only in flashcard mode.)
  • Toggle Options - Check these boxes to further customize your view:
    • Streak and Points Display - Show or hide information about your score streak and points.
    • On'yomi and Kun'yomi Details - Display or hide readings for kanji.

2. Game Panel

This panel displays the main gameplay area, where you’ll see flashcards or the input field for typing mode, based on your selected game mode.

3. Score Panel

Here you can track your score throughout the game. If you prefer a cleaner screen, you can hide this panel from the options.
